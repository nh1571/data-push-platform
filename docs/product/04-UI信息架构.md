# 管理台 UI 信息架构（一期）

> 依据：[[01-需求调研]]、[[02-领域模型]]、[[03-架构设计]]。  
> 一期使用者：**全管理员**；桌面优先（≥1280）。  
> 目标：能完成 A–F 闭环的配置与观测，而不是营销站。

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **配置 / 执行分离** | 左边管「定义」（数据源、通道、任务），执行页管「发生了什么」 |
| **任务是中心** | PushJob 聚合取数、渲染、通道、调度；少在全局散落入口 |
| **可行动的失败** | 日志页必须能重跑、看错误，而不是只堆 JSON |
| **试跑优先** | 数据源测连、通道试发、任务试跑路径短 |
| **危险可逆提示** | 删除、停用、对真实通道「立即推送」需确认 |
| **密钥脱敏** | 连接串/Token 只显示掩码，改则整段重填 |

---

## 2. 站点地图（导航）

```text
工作台（可选，一期可做极简）
数据源
通道
推送任务
  └─ 任务详情（子页，含试跑入口）
执行记录
系统
  └─ API Token（给 DS 的机器凭证）
  └─ 个人/登录（极简）
```

**侧栏一级（建议固定 5 项）：**

| 导航 | 路由示意 | 对应能力 |
|------|----------|----------|
| 工作台 | `/` | 最近失败、今日 Run 数（可二期加厚） |
| 数据源 | `/data-sources` | A |
| 通道 | `/channels` | C |
| 推送任务 | `/push-jobs` | 配置中枢 |
| 执行记录 | `/job-runs` | F、观测 |
| 系统 | `/settings` | 机器 Token 等 |

不单独做「渲染模板库」一级导航：渲染配置挂在 **任务** 上（一期够用；模板复用二期再抽）。

## 3. 关键用户路径

### 3.1 新建一条可跑通的推送（首次）

```text
1 数据源 → 新建 → 填 Doris/MySQL → 测试连接 → 保存
2 通道   → 新建钉钉通道 → 填配置 → 试发（固定文案）→ 保存
3 推送任务 → 新建
     - 选数据源 + SQL
     - 配渲染（文本/图/卡片/文件，可多选）
     - 选通道
     - skip_if_empty、调度（可选）
4 任务详情 → 立即执行（试跑）
5 执行记录 → 看状态 / 失败原因
```

### 3.2 日常：改 SQL 后验证

```text
推送任务 → 打开任务 → 改 SQL → 保存 → 立即执行 → 执行记录
```

### 3.3 DS 对接

```text
系统 → 创建机器 Token → 复制
在 DS 配 HTTP：POST /api/v1/push-jobs/{id}/run + Token
本系统执行记录中 trigger=external_http 可区分
```

### 3.4 失败处理

```text
执行记录 → 筛选 failed/partial → 详情看步骤日志
→ 重跑（新 JobRun，用最新配置）
```

## 4. 页面说明

### 4.1 工作台 `/`（一期可极简）

| 区块 | 内容 |
|------|------|
| 今日摘要 | JobRun 成功/失败/部分失败数 |
| 最近失败 | 最近 N 条 failed/partial，点进详情 |
| 快捷入口 | 新建任务、执行记录 |

一期可只做「最近失败 + 链到任务/记录」，统计可后补。

---

### 4.2 数据源列表 `/data-sources`

| 元素 | 说明 |
|------|------|
| 表格列 | 名称、类型(mysql/doris)、主机摘要、状态、更新时间 |
| 操作 | 新建、编辑、测试连接、删除（确认） |
| 空态 | 引导「先创建数据源」 |

### 4.3 数据源表单（新建/编辑）

| 字段 | 说明 |
|------|------|
| 名称 | 展示用 |
| 类型 | mysql / doris |
| 主机/端口/库/用户 | |
| 密码 | 脱敏；编辑时「留空表示不修改」 |
| 额外参数 | JSON 或键值（可选） |
| 操作 | 测试连接、保存 |

---

### 4.4 通道列表 `/channels`

| 元素 | 说明 |
|------|------|
| 列 | 名称、类型(dingtalk)、目标摘要、更新时间 |
| 操作 | 新建、编辑、试发、删除 |

### 4.5 通道表单

| 字段 | 说明 |
|------|------|
| 名称 | |
| 类型 | 一期固定/仅 dingtalk |
| 凭证与目标 | 按插件表单动态渲染（Webhook/AppKey 等） |
| 操作 | 试发（固定「测试消息」）、保存 |

试发成功/失败 Toast + 可查最近试发结果（可只 Toast）。

---

### 4.6 推送任务列表 `/push-jobs`

| 元素 | 说明 |
|------|------|
| 列 | 名称、数据源、通道数、调度摘要、启用状态、最近 Run 状态 |
| 筛选 | 启用/停用、关键字 |
| 操作 | 新建、进入详情、启用/停用、立即执行（二次确认） |

### 4.7 任务详情/编辑 `/push-jobs/:id`

**建议单页分区（锚点或 Tabs）：**

| Tab/区 | 内容 |
|--------|------|
| **基本信息** | 名称、描述、启用、`skip_if_empty` |
| **取数** | 数据源、SQL 编辑器、参数说明（如 biz_date） |
| **渲染** | 多部件配置：启用哪些 Renderer + 各自配置（模板、图表类型、导出格式等） |
| **通道** | 多选已有通道（顺序可选） |
| **调度** | 开关 + Cron + 时区；说明「也可由 DS HTTP 触发」 |
| **运行** | 立即执行、最近 5 次 Run 摘要表（链到执行详情） |

**SQL 区 UX：** 等宽字体、行号；可选「仅校验语法」不做（一期可不做）；试跑走「立即执行」。

**渲染区 UX：** 四个形态用 Checkbox/卡片选择；选中后展开配置面板；未选中的不生成对应 MessagePart。

---

### 4.8 执行记录列表 `/job-runs`

| 元素 | 说明 |
|------|------|
| 筛选 | 时间范围、PushJob、状态(pending/running/succeeded/failed/partial)、触发类型 |
| 列 | 时间、任务名、触发、状态、耗时、操作 |
| 操作 | 详情、重跑（确认：将用**最新任务配置**） |

状态用颜色徽章：成功绿、失败红、部分橙、运行蓝。

### 4.9 执行详情 `/job-runs/:id`

| 区块 | 内容 |
|------|------|
| 头信息 | 任务名、状态、触发类型、操作者/DS、起止时间、parent_run（若重跑） |
| 步骤时间线 | 取数 → 渲染 → 各通道投递（成功/失败/skipped） |
| 取数摘要 | 行数、耗时；过大不展示全表，可「预览前 20 行」 |
| 消息预览 | 文本折叠展示；图片缩略图；文件下载链；卡片 JSON/预览 |
| 配置快照 | 只读展开（审计用） |
| 操作 | 重跑、回到任务编辑 |

---

### 4.10 系统 `/settings`

| 子页 | 内容 |
|------|------|
| API Token | 列表、创建（名+Token 仅创建时展示一次）、吊销 |
| 关于/版本 | 可选 |

一期不做用户管理 UI（全管理员，登录即可）。

## 5. 全局 UX 细节

| 项 | 约定 |
|----|------|
| 布局 | 左侧导航 + 顶栏（用户/退出）+ 内容区 |
| 语言 | 中文界面 |
| 表单校验 | 必填即时；保存失败展示服务端错误 |
| 列表分页 | 默认 20 条 |
| 确认框 | 删除；立即执行（文案标明将发真实通道）；重跑 |
| 权限 | 一期所有登录用户同权；菜单不做隐藏 |

## 6. 页面与领域/能力映射

| 页面 | 领域对象 | 成功标准 |
|------|----------|----------|
| 数据源 | DataSource | A |
| 通道 | Channel | C |
| 推送任务 | PushJob, QuerySpec, RenderSpec, Schedule | A–D 配置面 |
| 执行记录 | JobRun, Delivery | F、E 可观测 |
| 系统 Token | ExternalTrigger 凭证 | E |

---

## 7. 一期明确不做的 UI

- 角色权限配置
- 渲染模板中心（全局复用库）
- 可视化拖拽编排（DS 级工作流画布）
- 移动端适配
- 消息内容富文本所见即所得（SQL/模板以代码编辑为主即可）

---

## 8. UI 开放项结论（2026-07-16，U1–U3 均按默认）

| ID | 最终结论 |
|----|----------|
| **U1** | 一期保留 **极简工作台**（最近失败 + 摘要入口） |
| **U2** | 任务编辑使用 **Tabs**（基本 / 取数 / 渲染 / 通道 / 调度 / 运行） |
| **U3** | SQL 一期使用 **等宽 textarea**；语法高亮编辑器后置 |
