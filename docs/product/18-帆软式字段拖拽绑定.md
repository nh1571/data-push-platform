# 帆软式字段拖拽绑定

## 用户决策（2026-07-17）

> 内容工作台绑定方式 **偏帆软字段拖拽**（不是只点选下拉/点芯片）。

## 交互模型

```text
取数 → 字段列表（按数据集）
  → 拖字段到组件槽位
       Text/Alert  → 插入 {{列名}}，并绑定数据集
       KPI       → 数值列
       Chart     → 分类槽 / 数值槽（明确两个 drop zone）
       Table     → 绑定整表数据集
  → 编译预览
```

## 与旧点击芯片的关系

- 点击字段 = 仍可用（快捷）
- **主路径 = 拖拽到槽位**（槽位在拖拽时高亮）

## 不做

- 单元格级类 Excel 交叉表绑定（二期）
- 无限自由画布坐标

## 变更

- 2026-07-17：用户确认偏帆软字段拖拽；开始落地 UI
