# 购物车式组件交互

## 用户要求（2026-07-17）

- 组件步逻辑要 **简单清爽**，看不懂复杂绑定槽/拖拽。
- 流程：一个一个 **生成组件 → 加入清单（购物车）→ 下一步组装**。
- **组件步**：只管「造组件」（类型 + 数据字段 + 基础样式）。
- **组装步**：只管位置顺序、大小、配色等版式。

## 四步（精简）

| 步 | 做什么 | 不做什么 |
|----|--------|----------|
| 1 数据 | 取数、字段列表 | 不排版 |
| 2 组件购物车 | 选型→选字段（下拉）→加入清单；可改样式标签 | 不拖布局 |
| 3 组装 | 顺序、宽度、颜色 | 不改 SQL/字段映射（可链回组件） |
| 4 预览 | 编译成图/试推 | — |
